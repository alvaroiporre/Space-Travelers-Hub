# ğŸ“– Space Travelers Hub! <a name="about-project"></a>

**Space Travelers Hub!** This is a web application for a company that provides commercial and scientific space travel services. The application will allow users to book rockets and join selected space missions. The Space Travelers' Hub consists of Rockets, Missions, and the My Profile section.

### Key Features <a name="key-features"></a>

- **This is a react group project**
- **The Space Travelers' Hub consists of Rockets, Missions, and the My Profile section.**
- **The Rockets section displays a list of all available SpaceX rockets.**
- **Users can book each rocket by clicking the reservation button or cancel the previously made booking.**
